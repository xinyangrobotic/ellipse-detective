Original author: mikispace (https://sourceforge.net/projects/yaed/)

### How to compile:
编译过程，通过cmake的方式对程序进行编译,首先进入程序的根目录
```sh
mkdir build
cd build
cmake ..
make -j4
```

### How to run:

```sh
./fast_ellipse_detector
```
